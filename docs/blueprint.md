# **App Name**: InventoryEdge

## Core Features:

- Dashboard: Display key inventory metrics and system status at a glance.
- Inventory Management: Create, read, update, and delete inventory items with detailed information (name, description, quantity, location, etc.).
- Inventory List and Filters: Show list of items and the option to filter by category, date added, availability, and manufacturer.
- AWS RDS Connection Module: Display the current connection status to the AWS RDS database.
- Sidebar Navigation: Collapsible sidebar for easy navigation between modules.
- Alerts and Notifications: Send automatic alert of out-of-stock items, expired or near to expire products.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey stability and trust.
- Background color: Light gray (#F0F2F5), offering a clean and modern backdrop.
- Accent color: Teal (#009688) to highlight interactive elements and important information.
- Body and headline font: 'Inter', a sans-serif font for clear and accessible information display.
- Use a set of consistent and professional icons that match the app's aesthetic.
- Maintain a clean and organized layout to present the most important information at a glance.
- Subtle transitions and animations to create a dynamic, user-friendly experience.
